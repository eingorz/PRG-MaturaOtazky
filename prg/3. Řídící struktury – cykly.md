Řídící struktury určují pořadí, v jakém se vykonávají příkazy v programu.  
Jedním ze základních typů řídících struktur jsou **cykly**.

Cyklus umožňuje **opakovaně provádět určitou část programu**, tedy lépe logicky zapsat skok zpět v programu tak aby se daná skupina kroků vykonala vícekrát

Používají se například pro:
- opakované výpočty
- náhardu za rekurzi
- procházení prvků pole nebo seznamu
- načítání vstupů od uživatele
### 3.1 Princip cyklu
Cyklus opakuje množinu kroků většinou za nějaké podmínky, jelikož bez ní bychom mohli vytvořit pouze cyklus, který se neprovede nikdy, nebo cyklus, který se bude provádět navždy (Nekonečný cyklus)

Nekonečné cykly většinou lze ovšem přerušit i vložením podmínky pro vyskočení z cyklu dovnitř těla cyklu

Podle počáteční podmínky cyklu se cykly dělí na:
### 3.1.1 Cyklus s podmínkou na začátku

Podmínka se vyhodnocuje před vykonáním těla cyklu a určuje zda se tělo cyklu vůbec provede,
po vykonání těla cyklu se řízení vrací opět před tuto podmínku. Pokud tato podmínka selže
záporná větev cyklu přesměruje řízení za cyklus

Př.:
```C
    while (x < 0) {
        // tělo cyklu
        x++; // zajištění konečnosti cyklu
    }
```

## 3.1.2 Cyklus s podmínkou na konci

Podmínka se vyhodnocuje po prvním vykonání těla cyklu a určuje zda se má tělo cyklu vykonat znovu.

Př.:
```C
    do {
        // tělo cyklu
        x++; // zajištění konečnosti cyklu
    } while (x < 0);
```

Tělo cyklu se v tomto případě provede nejméně jednou a hodnota x se inkrementuje nezávisle
na hodnotě podmínky `x < 0`
## 3.1.3 Cyklus s řídící proměnnou / Cyklus s určitým počtem opakování

Cyklus s řídící proměnnou většinou zjednodušuje užití proměnné k zajištění určitého počtu opakování cyklu.
Hodnota aktuálního opakování taktéž díky tomuto lze použít v kódu.

Př.:
```C
    int i;
    for (i = 0; i < 10; i++) {
        // tělo cyklu (vykonáno přesně 10x)
    }
```

Konkrétně cyklus s řídící proměnnou `for`, který se řadí do cyklů s podmínkou na začátku, se skládá ze tří částí:

Př.:
```C
    for (/* inicializátor */; /* podmínka */; /* inkrement */) {
        // tělo cyklu
    }
```
#### Inicializace / Inicializátor
Je instrukce, která se vykoná před prvotním vyhodnocení podmínky na začátku.
#### Podmínka
Je výraz, jehož pravdivost určuje zda se tělo vykoná či nikoliv
#### Inkrement
Je instrukce, která se vykoná po každém vykonání těla cyklu

*Poznámka: cyklus `for` nevyžaduje striktně použití proměnné, tyto jeho tři složky (instrukce) se vyhodnocují nezávisle na tom zda použijeme proměnnou například zde vidíme cyklus `for` využit k iteraci řetězce bez použití proměnné pro počet iterací*

Př.:
```C

    char c;

    char *str = "Muj retezec";

  

    for (;(c = *str) != '\0'; str++) {

        // tělo cyklu ve kterém mohu použít proměnnou `c` k práci s aktuálně iterovaným znakem řetězce

    }

```
## 3.1.4 Nekonečný cyklus

Jako nekonečný cyklus označujeme cyklus, jehož tělo se bude provádět nekonečněkrát.
Většinou je cyklus z principu nekonečný jelikož nic v těle cyklu neovlivňuje pravdivost podmínky cyklu.

Př.:
```C
    while (1) {
        // tělo cyklu
    }
```

Tělo nekonečného cyklu lze ovšem i tak opustit a to vyskočením z něj.
Většina jazyků toto umožňuje pomocí klíčového slova `break`.
## 4. Break a Continue
Klíčové slovo `break` slouží jako instrukce která vyskočí z těla cyklu za něj (Tedy do záporné větve podmínky cyklu)

Klíčové slovo `continue` slouží jako instrukce která přeskočí zbytek těla cyklu a skočí přímo před podmínku cyklu (Nebo před inkrementační instrukci v případě cyklu s řídící proměnnou)

Př.:
```C
    int x = 5;
    while (1) {
        // tělo cyklu
        x--;
        if (x <= 0) break; // v případě že je x menší nebo rovno nule, vyskakujeme z těla cyklu
        if (x == 3) continue; // v případě že je x rovno třem, zbytek těla cyklu se přeskakuje
        
        // tato část těla cyklu se vykoná v případě, že x je větší než nula a nerovno třem
    }
```

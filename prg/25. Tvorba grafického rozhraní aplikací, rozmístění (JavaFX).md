# 1. Soubory pro tvorbu JavaFX Application

JavaFX aplikace se obvykle skládá z několika typů souborů:

- Java třídy (.java)
    - Obsahují logiku aplikace
    - Hlavní třída dědí od Application
    - Přepisuje metodu start(Stage stage)
- FXML soubory (.fxml)
    - XML soubory pro definici grafického rozhraní
    - Oddělují vzhled od logiky
    - Používají se spolu s kontrolerem
- Controller (řadič)
    - Java třída propojená s FXML
    - Obsahuje obsluhu událostí (např. kliknutí na tlačítko)
- CSS soubory
    - Slouží ke stylování aplikace (barvy, fonty, velikosti)

# 2. Základní pojmy – Stage, Scene, Node
## Stage
Představuje okno aplikace
Je hlavní kontejner (např. hlavní okno programu)
V aplikaci může být více oken

```Java
stage.setTitle("Moje aplikace");
stage.show();
```

## Scene
Obsahuje veškerý vizuální obsah
Každý Stage má jednu Scene

```Java
Scene scene = new Scene(root, 400, 300);
stage.setScene(scene);
```

## Node
Základní stavební prvek GUI
Každý prvek (tlačítko, text, panel) je Node
Všechny prvky tvoří stromovou strukturu

Příklady:
- Button
- Label
- TextField
- Pane

# 3. Rozložení (Layout)

Rozložení určuje, jak jsou prvky umístěny v okně.

Nejčastější layouty:
## Pane
- Základní kontejner
- Prvky mají pevně dané souřadnice

## HBox
- Prvky vedle sebe (vodorovně)

```Java
HBox hbox = new HBox(10); // mezera 10 px
hbox.getChildren().addAll(btn1, btn2);
```

## VBox
- Prvky pod sebou (svisle)

## BorderPane
- Rozdělení na 5 částí:
    `top`, `bottom`, `left`, `right`, `center`

## GridPane
- Rozložení do tabulky (řádky a sloupce)

```Java
grid.add(button, 0, 0); // sloupec, řádek
```

## FlowPane
Prvky se automaticky zalamují (jako text)

# 4. Práce s komponentami

Komponenty (ovládací prvky) slouží k interakci s uživatelem.

Základní komponenty:
- Button – tlačítko
- Label – text
- TextField – vstupní pole
- CheckBox – zaškrtávací pole
- RadioButton – výběr jedné možnosti
- ComboBox – rozbalovací seznam
- Obsluha událostí

## Reakce na akce uživatele (např. kliknutí):

```Java
button.setOnAction(e -> {
    System.out.println("Kliknuto!");
});
```

## Práce s hodnotami

```Java
String text = textField.getText();
label.setText("Zadal jsi: " + text);
```

## Přidávání komponent do layoutu
```Java
VBox root = new VBox();
root.getChildren().addAll(label, textField, button);
```

  
## Znakový typ `char`  
- Ordinální datový typ **`char`** slouží pro ukládání **jednoho znaku**.  
- Standardně zabírá **1 byte (8 bitů)**.  
- Hodnota znaku je ve skutečnosti **číselná hodnota podle ASCII tabulky**.  
  
Příklad:  
```c  
char c = 'A';
```

Interně:
```c
'A' = 65 (ASCII)
```

Proto lze se znaky provádět i **aritmetické operace**.
```c
char c = 'A';  
c = c + 1;  
  
printf("%c", c); // B
```

---
## ASCII reprezentace

C používá znaky podle **ASCII (American Standard Code for Information Interchange)**.

Příklady:

| znak | ASCII |
| ---- | ----- |
| A    | 65    |
| a    | 97    |
| 0    | 48    |

To umožňuje například převod znaků na čísla:
```c
char digit = '5';  
int value = digit - '0'; // 5
```

---
## Escape sekvence

Speciální znaky zapisované pomocí **escape sekvencí**.

|sekvence|význam|
|---|---|
|`\n`|nový řádek|
|`\t`|tabulátor|
|`\\`|zpětné lomítko|
|`\"`|uvozovky|
|`\0`|nulový znak|

Příklad:
```c
printf("Ahoj\nSvete");
```

# Řetězce v jazyce C

## Definice řetězce

V jazyce C **neexistuje speciální datový typ string**.

Řetězec je pouze:
**pole znaků (`char[]`) zakončené nulovým znakem `'\0'`**

Tento znak označuje **konec řetězce**.

---
## Deklarace řetězce

### Statické pole znaků
```c
char str[6] = {'H','e','l','l','o','\0'};
```

---
### Zkrácený zápis

```c
char str[] = "Hello";
```

Kompilátor automaticky přidá **nulový znak**.

Paměť:
`H e l l o \0`

---
## Velikost řetězce

Velikost pole:
```c
char str[] = "Hello";  
  
sizeof(str) // 6
```

Počet znaků (bez `\0`): 5

---
## Přístup ke znakům

Řetězec je pole, takže lze použít index.

```c
char str[] = "Hello";  
  
printf("%c", str[1]); // e
```

---
## Řetězec jako pointer

Řetězec lze reprezentovat také pointerem.
```c
char *str = "Hello";
```

Rozdíl:
### Pole znaků
```c
char str[] = "Hello";  
  
str[0] = 'h'; // OK
```

### Pointer na literál
```c
char *str = "Hello";  
  
str[0] = 'h'; // undefined behavior
```

Literál je uložen v **read-only paměti**. Na většině operačních systémů, zasahování do této části paměti = segmentační chyba.

---
## Iterace přes řetězec

Pomocí indexu:
```c
for(int i = 0; str[i] != '\0'; i++){  
    printf("%c", str[i]);  
}
```

Pomocí pointeru:
```c
char *p = str;  
  
while(*p != '\0'){  
    printf("%c", *p);  
    p++;  
}
```

# Knihovna `string.h`

Standardní knihovna pro práci s řetězci.

`#include <string.h>`

Obsahuje funkce pro:
- délku řetězce
- kopírování
- porovnání
- spojování
- vyhledávání

---
## `strlen`

Vrací **délku řetězce bez nulového znaku**.
```c
size_t strlen(const char *str);
```

Příklad:
```c
char str[] = "Hello";  
  
printf("%lu", strlen(str)); // 5
```

---
## `strcpy`

Kopíruje řetězec.
```c
char *strcpy(char *dest, const char *src);
```

Příklad:
```c
char src[] = "Hello";  
char dest[10];  
  
strcpy(dest, src);
```

---
## `strncpy`

Bezpečnější kopie s omezením délky.
```c
char *strncpy(char *dest, const char *src, size_t n);
```

---
## `strcmp`

Porovnání dvou řetězců.
```c
int strcmp(const char *s1, const char *s2);
```

Výsledek:

| návratová hodnota | význam              |
| ----------------- | ------------------- |
| 0                 | řetězce jsou stejné |
| <0                | s1 < s2             |
| >0                | s1 > s2             |

Příklad:
```c
if(strcmp(a, b) == 0){  
    printf("shodne");  
}
```

---
## `strncmp`

Porovnání prvních **n znaků**.
```c
strncmp(s1, s2, n);
```

---
## `strcat`

Spojení dvou řetězců.
```c
char *strcat(char *dest, const char *src);
```

Příklad:
```c
char a[20] = "Hello ";  
char b[] = "World";  
  
strcat(a, b);
```

Výsledek:
`Hello World`

---
## `strncat`

Spojení s omezením počtu znaků.
```c
strncat(dest, src, n);
```

---
## `strchr`

Vyhledá první výskyt znaku.
```c
char *strchr(const char *str, int c);
```

---
## `strstr`

Vyhledá podřetězec.
```c
char *strstr(const char *haystack, const char *needle);
```

# Typické chyby

## Chybějící nulový znak
```c
char str[5] = {'H','e','l','l','o'}; // chyba
```

---

## Přetečení bufferu
```c
char str[5];  
  
strcpy(str, "Hello world"); // buffer overflow
```

---

## Modifikace string literal
```c
char *str = "Hello";  
  
str[0] = 'h'; // undefined behavior
```

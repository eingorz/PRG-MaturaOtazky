## Základní princip práce se soubory  
  
Soubor je **sekvence dat uložená na sekundární paměti (disku)**.    
Programy používají soubory pro:  
- **trvalé ukládání dat**  
- výměnu dat mezi programy  
- zpracování velkých objemů dat  
  
Práce se souborem obvykle probíhá v krocích:  
1. otevření souboru  
2. čtení nebo zápis dat  
3. uzavření souboru  
  
---  
# Soubory v jazyce C  
  
Pro práci se soubory se používá **standardní knihovna `stdio.h`**.  
  
Soubor je reprezentován strukturou:  
```c  
FILE
```

Ukazatel na soubor:
```c
FILE *f;
```

---
# Otevření souboru

Soubor se otevírá funkcí:
```c
FILE *fopen(const char *filename, const char *mode);
```

Příklad:
```c
FILE *f = fopen("data.txt", "r");
```

Pokud otevření selže, funkce vrací `NULL`

Kontrola:
```c
const char *filename = "data.txt"
FILE *f = fopen(filename, "r");  
  
if (f == NULL) { 
	fprintf(stderr, "Error reading file \"%s\"\n", filename);
    perror("fopen");
}
```

Výstup na chybový proud v případě např. že soubor neexistuje:
```
Error reading file "data.txt"
fopen: No such file or directory 
```

---
# Režimy otevření souboru

## Textové režimy

### Read (`r`)
- otevře soubor **pouze pro čtení**
- soubor **musí existovat**
- čtení začíná **od začátku souboru**

```c
FILE *f = fopen("data.txt", "r");
```

---
### Write (`w`)
- otevře soubor **pro zápis**
- pokud soubor existuje → **je přepsán**
- pokud neexistuje → **je vytvořen**

```c
FILE *f = fopen("data.txt", "w");
```

---
### Append (`a`)
- zápis **na konec souboru**
- soubor se **nepřepisuje**
- pokud neexistuje → vytvoří se
- 
```c
FILE *f = fopen("data.txt", "a");
```

---
# Binární režimy

Používají se pro **binární data** (např. struktury, obrázky).

|režim|význam|
|---|---|
|`rb`|čtení binárního souboru|
|`wb`|zápis binárního souboru|
|`ab`|přidávání do binárního souboru|

Příklad:
```c
FILE *f = fopen("data.bin", "rb");
```

Rozdíl mezi textovým a binárním režimem:
- textový režim může provádět **konverzi znaků (např. newline)**
- binární režim zapisuje **data přesně tak, jak jsou v paměti**

---
# Zavření souboru

Soubor je nutné po práci uzavřít:
```c
int fclose(FILE *stream);
```

Příklad:
```c
fclose(f);
```

Důvody:
- uvolnění systémových prostředků
- zajištění zápisu bufferu na disk

---
# Čtení ze souboru

## `fscanf`

Čtení formátovaných dat.
```c
fscanf(f, "%d %f", &a, &b);
```

---
## `fgets`

Čtení jednoho řádku.
```c
char buffer[100];  
  
fgets(buffer, sizeof(buffer), f);
```

---
## `fgetc`

Čtení jednoho znaku.
```c
int c = fgetc(f);
```

---
# Zápis do souboru

## `fprintf`

Formátovaný zápis.

```c
fprintf(f, "%d %f\n", a, b);
```

---
## `fputs`

Zápis řetězce.

```c
fputs("Hello\n", f);
```

---
## `fputc`

Zápis jednoho znaku.

```c
fputc('A', f);
```

---
# Binární čtení a zápis

Používají se funkce:
- `fread`
- `fwrite`

---
## `fread`

```c
size_t fread(void *ptr, size_t size, size_t count, FILE *stream);
```

Příklad:
```c
fread(&x, sizeof(int), 1, f);
```

---

## `fwrite`
```c
size_t fwrite(const void *ptr, size_t size, size_t count, FILE *stream);
```

Příklad:
```c
fwrite(&x, sizeof(int), 1, f);
```

---
# Pozice v souboru

## `fseek`

Posun v souboru.

```c
fseek(f, offset, SEEK_SET);
```

Možné hodnoty:

|konstanta|význam|
|---|---|
|`SEEK_SET`|začátek souboru|
|`SEEK_CUR`|aktuální pozice|
|`SEEK_END`|konec souboru|

---
## `ftell`

Vrací aktuální pozici.

```c
long pos = ftell(f);
```

---
## `rewind`

Návrat na začátek souboru.

```c
rewind(f);
```

# Soubory v jazyce Java

Java poskytuje **objektově orientovaný přístup k práci se soubory**.

Balíčky:
```Java
java.io  
java.nio
```

---

# Třída `File`

Reprezentuje soubor nebo adresář.
```Java
File f = new File("data.txt");
```

Příklady metod:
- `exists()`
- `createNewFile()`
- `delete()`
- `length()`

---
# Čtení souborů v Java

## `FileReader`

Čtení textového souboru.
```Java
FileReader fr = new FileReader("data.txt");
```

---
## `BufferedReader`

Efektivní čtení po řádcích.
```Java
BufferedReader br = new BufferedReader(new FileReader("data.txt"));  
  
String line = br.readLine();
```

Příklad:
```Java
BufferedReader br = new BufferedReader(new FileReader("data.txt"));  
  
String line;  
while((line = br.readLine()) != null){  
    System.out.println(line);  
}  
  
br.close();
```

---
# Zápis do souboru

## `FileWriter`
```Java
FileWriter fw = new FileWriter("data.txt");
```

---
## `BufferedWriter`
```Java
BufferedWriter bw = new BufferedWriter(new FileWriter("data.txt"));  
  
bw.write("Hello");  
bw.newLine();  
  
bw.close();
```

---

# Uzavření čtecích/zápisových objektů

Po ukončení operace je vždy nutno objekt třídy `FileReader`/`BufferedReader` nebo `FileWriter`/`BufferedWriter` uzavřít metodou `close()` rozhraní `Closeable`.

Jelikož rozhraní `Closeable` využívá dědičnosti ke sdílení chování s rozhraním `AutoCloseable` je objekt tříd `FileReader`/`BufferedReader` nebo `FileWriter`/`BufferedWriter` schopen se sám uzavřít při destrukci nebo při jeho užití v `try` bloku takto:

```Java
try (FileWriter fw = new FileWriter("data.txt")) {
	// práce se souborem
} // fw se automaticky uzavře
```

# Moderní práce se soubory (Java NIO)

Balíček:

java.nio.file

Třídy:
- `Path`
- `Files`

Příklad čtení:

Path path = Path.of("data.txt");  
  
String content = Files.readString(path);

```Java
Files.writeString(path, "Hello world");
```

# Rozdíl mezi C a Java

|vlastnost|C|Java|
|---|---|---|
|přístup|procedurální|objektový|
|reprezentace souboru|`FILE*`|objekty (`File`, `Path`)|
|práce s pamětí|manuální|automatická (GC)|
|bezpečnost|nízká|vyšší|
|API|`stdio.h`|`java.io`, `java.nio`|
